Welcome to NodeConductor's documentation!
=========================================

NodeConductor is a RESTful server for management of the IT infrastructure. It's using a project-based approach for
separation of the managed resources (VMs, subnets, applications and more).

Features include:

- support for multiple IaaS backends;
- flexible user, group and project management;
- tag-based grouping of resources into environments;
- templates based on configuration management approach;
- orchestration with SaltStack;
- used license reporting;
