Structure events
++++++++++++++++

.. glossary::

    **customer_creation_succeeded**
        Customer has been created.

    **customer_update_succeeded**
        Customer has been updated.

    **customer_deletion_succeeded**
        Customer has been deleted.

        User has been removed from organization.
